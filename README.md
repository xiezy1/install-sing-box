# 使用方法
下载脚本 设置权限 执行
```shell
sudo wget https://raw.githubusercontent.com/xiezy1/install-sing-box/main/install.sh && \
sudo chmod +x install.sh && \
sudo ./install.sh
```

# 修改订阅地址

编辑update_subscript.sh
```shell
# 这里替换自己的订阅地址
VAR1=$(curl -s https://example.com/your-subscription-url)
```
